1. 簡介

SmartCard（以下簡稱「本應用程式」）是一款協助您管理會員卡條碼、並自動偵測附近門市的 Android 應用程式。本隱私權政策說明我們如何收集、使用及保護您的個人資料。

2. 收集的資料

2.1 位置資料（GPS）

  • 收集目的：偵測您附近的門市，自動顯示對應的會員卡條碼。
  • 收集方式：透過 Android 系統的精確位置（Fine Location）、概略位置（Coarse Location）及背景位置（Background Location）權限。
  • 儲存方式：位置資料不會上傳至任何伺服器，僅在裝置本地進行運算與比對。
  • 保留期限：位置快取暫存於裝置本地，可隨時透過清除應用程式資料移除。

2.2 會員卡資料

  • 收集內容：您手動輸入或掃描的會員卡名稱、條碼數值、條碼格式。
  • 儲存方式：所有會員卡資料以 AES-256 加密儲存於裝置本地，不會上傳至任何雲端服務。
  • 保留期限：資料保留至您主動刪除或解除安裝應用程式為止。

2.3 診斷與分析資料 Firebase

以下資料由 Google Firebase 處理，適用 Google 隱私權政策。
  • Firebase Analytics：收集匿名的應用程式使用統計（功能使用頻率、工作階段長度）。不包含任何個人識別資訊。
  • Firebase Crashlytics：收集應用程式崩潰報告（裝置型號、Android 版本、錯誤堆疊）。Debug 版本自動停用此功能。

3. 資料分享

本應用程式不會將您的個人資料販售、出租或分享給第三方，以下情況除外：

  • Firebase（僅限匿名診斷資料，如上述 2.3）
  • 法律要求之情形

4. 資料安全

  • 會員卡資料使用 AES-256-CBC 加密，金鑰透過 Android Keystore 保護。
  • 位置資料僅於裝置本地處理，不經過任何外部網路傳輸。

5. 兒童隱私

本應用程式不針對 13 歲以下兒童設計，亦不刻意收集兒童個人資料。

6. 您的權利

您可隨時：

  • 透過 Android 系統設定撤銷位置權限
  • 在應用程式內刪除所有會員卡資料
  • 解除安裝應用程式以移除所有本地資料

7. 政策變更

若本政策有重大變更，將於 GitHub 頁面公告，並更新本文件的「最後更新日期」。

1. Introduction

SmartCard is an Android application that helps you manage membership card barcodes and automatically detects nearby stores. This Privacy Policy explains how we collect, use, and protect your personal information.

2. Information We Collect

2.1 Location Data (GPS)

  • Purpose: To detect nearby stores and automatically display the relevant membership card barcode.
  • Method: Via Android Fine Location, Coarse Location, and Background Location permissions.
  • Storage: Location data is never uploaded to any server. All processing is done locally on your device.
  • Retention: Location cache is stored locally and can be removed by clearing app data at any time.

2.2 Membership Card Data

  • Content: Card names, barcode values, and barcode formats you enter or scan.
  • Storage: All card data is encrypted with AES-256 and stored locally on your device. No data is uploaded to any cloud service.
  • Retention: Data is retained until you delete it or uninstall the app.

2.3 Diagnostic and Analytics Data Firebase

The following data is processed by Google Firebase, subject to Google's Privacy Policy.
  • Firebase Analytics: Collects anonymous app usage statistics (feature usage frequency, session length). No personally identifiable information is included.
  • Firebase Crashlytics: Collects crash reports (device model, Android version, error stack traces). Automatically disabled in debug builds.

3. Data Sharing

We do not sell, rent, or share your personal data with third parties, except:

  • Firebase (anonymous diagnostic data only, as described in 2.3)
  • As required by law

4. Data Security

  • Membership card data is encrypted with AES-256-CBC, with keys protected by Android Keystore.
  • Location data is processed entirely on-device and is never transmitted over any network.

5. Children's Privacy

This app is not designed for children under 13, and we do not knowingly collect personal information from children.

6. Your Rights

You may at any time:

  • Revoke location permissions via Android system settings
  • Delete all membership card data within the app
  • Uninstall the app to remove all local data

7. Policy Changes

Material changes to this policy will be announced on the GitHub page, and the "Last Updated" date will be revised accordingly.
