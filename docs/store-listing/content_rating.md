# IARC 內容分級問卷答案

以下為 Google Play Console 內容分級問卷的建議回答：

## 類別
工具 / 生活品味

## 問卷回答

| 問題 | 回答 |
|------|------|
| 是否包含暴力內容？ | 否 |
| 是否包含性或情色內容？ | 否 |
| 是否涉及受管制物質（藥物、酒精、菸草）？ | 否 |
| 是否包含粗俗用語？ | 否 |
| 是否允許使用者互動或分享內容？ | 否 |
| 是否分享使用者的位置資訊？ | 否（僅本地使用，不分享） |
| 是否包含購買功能？ | 否 |
| 是否包含廣告？ | 否 |
| 是否包含賭博內容？ | 否 |
