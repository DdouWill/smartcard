# SmartCard 商店資訊

## App 名稱
SmartCard 智慧會員卡

## 簡短說明（80 字以內）
自動偵測附近門市，桌面 Widget 即時顯示會員卡條碼。完全離線，不需網路。

## 完整說明

SmartCard 是一款智慧會員卡錢包，將你所有的會員卡條碼集中管理，並透過 GPS 自動偵測附近門市，在桌面小工具（Widget）即時顯示對應的會員卡。

▸ 核心功能
• 會員卡管理：新增、編輯、刪除，支援 CODE128、EAN13、QR Code 等多種條碼格式
• 條碼掃描：用相機掃描實體卡片，快速建立數位卡片
• 智慧定位：GPS 自動偵測你附近的門市，顯示最相關的會員卡

▸ 桌面 Widget
• 上下滑動切換多張卡片
• 依門市距離自動排序，最近的門市卡片排最前面
• 背景每 15 分鐘自動更新，走到哪裡都能即時顯示

▸ 門市資料
• 內建超過 22,000 間門市座標，涵蓋 38 個品牌
• 包含 7-ELEVEN、全家、萊爾富、OK 超商等便利商店
• 全聯、家樂福、大潤發等量販超市
• 屈臣氏、康是美、寶雅等藥妝百貨

▸ 隱私優先
• 完全離線運作，不需要網路連線
• 卡片資料只存在你的手機上，不會上傳到任何伺服器
• 位置資料僅用於門市匹配，不會被蒐集或分享

最低需求：Android 8.0 以上
