# 資料安全聲明（Data Safety）

Google Play Console 填寫指引：

## 資料蒐集與分享總覽

| 問題 | 回答 |
|------|------|
| 此應用程式是否會蒐集或分享任何必要的使用者資料類型？ | 是 |
| 蒐集的所有使用者資料是否都經過加密傳輸？ | 是（Firebase 走 HTTPS） |
| 是否提供使用者要求刪除資料的方式？ | 是（解除安裝即刪除所有本地資料） |

## 資料類型明細

### 位置
| 項目 | 回答 |
|------|------|
| 蒐集大略位置？ | 否 |
| 蒐集精確位置？ | 是 |
| 用途 | App 功能（門市匹配） |
| 是否分享給第三方？ | 否 |
| 是否為必要資料？ | 否（使用者可拒絕授權，App 仍可使用） |

### 應用程式活動
| 項目 | 回答 |
|------|------|
| 蒐集應用程式互動資料？ | 是（Firebase Analytics 匿名統計） |
| 用途 | 分析（改善 App） |
| 是否分享給第三方？ | 否 |

### 診斷資料
| 項目 | 回答 |
|------|------|
| 蒐集當機記錄？ | 是（Firebase Crashlytics） |
| 用途 | 分析（修復錯誤） |
| 是否分享給第三方？ | 否 |

### 不蒐集的類型
- 個人資訊（姓名、Email、地址、電話）
- 財務資訊
- 訊息內容
- 相片或影片
- 音訊
- 檔案
- 聯絡人
- 瀏覽紀錄
- 裝置識別碼
